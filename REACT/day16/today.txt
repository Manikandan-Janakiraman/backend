What is HOC?
    Hoc is a function. It takes component as a input  - give the return new enhance component.

HOC Syntax:

    const withText = (WrappedComponent) =>(props)=>{

        return <WrappedComponent {...props} />
    }

    export default withText;

    Explanation:
        props & WrappedComponent - Parameter
        {...props} - Container (all data stored in the container)
