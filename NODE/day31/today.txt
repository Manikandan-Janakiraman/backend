File System (FS)
    -> The FS Module in node js allow the backend to interact directly 
       with the operating system file.
    -> Used to create, read, update, delete (CRUD) and manage files, directories
    -> Node can access system level resources, it makes fs a powerful backend tool it works synchronous and asynchronous
    -> Asynchronous suitable for scalability purpose

    -> absolute path, relative path

       manage the extensions, it prevent hard core file location.
